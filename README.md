# enshu4_team-project1
チーム開発した日記アプリを原型に、個人的に手を加え成果物としてherokuを用いてデプロイしたものになります。
日記アプリ

テストアカウント　ユーザネーム：guest パスワード:1234

　機能
 
 　・ソーシャルログイン/ログアウト
 
 　・日記の投稿、編集、削除、一覧表示

 　・スケジュール表（google calender APIとの連携）
